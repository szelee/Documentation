CbeMobileServiceDeleteCustomerPhotocard Method

This FEWS triggers a deletion of Customer photocard record for a specified photocard. Input must contain customer identifier and customer photocard identifier.

Remarks

Deletes customer photocard.

VALIDATION

The following validation checks are made by the CBE on the data provided by the FEA. Failure to meet these checks will result in appropriate error messages as shown in the appropriate API guide.

  1. All mandatory fields are present.
  2. Customer Id must exist as a customer account.
  3. Photocard Id must exist and belong to the specified customer.
