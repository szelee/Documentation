ICbeTvmServiceUnlockCtr Method

This call is for the TVM FEA to request to the CBE that a CTR record is unlocked in the LSM database, where the CBE has previously locked it. This call is only to be used in exceptional cases to resolve CTR collection issues as required, so that any CTR records which are in suspense (for whatever reason) do not remain locked.

Remarks

ASSUMPTIONS

  1. It is assumed that this FEWS is only used by the TVM where an issue has arisen with CTR collection and the TVM wishes to force the CTR to be unlocked so that it can be retrieved from another TVM.

VALIDATIONS

  1. All mandatory fields are present.
  2. CTR and journey references should exist on the LSM database
