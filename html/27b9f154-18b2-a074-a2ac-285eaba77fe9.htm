CbeReferenceDataServiceGetTicketTypes Method

This call is part of the reference data and is available to all retail channels. It is used to obtain list of ticket types with additional details such as description, display name etc. This data is provided from IPTIS with supplementary information provided by NRE.

ASSUMPTIONS

  1. It is assumed FEA will keep ticket type list in a cache to lookup a ticket type typically by FTOT code as they are used in journey planning and purchase (season) ticket calls.
  2. It is responsibility of the FEA to keep a cache up-to-date i.e. by synchronizing the data on daily basis.
