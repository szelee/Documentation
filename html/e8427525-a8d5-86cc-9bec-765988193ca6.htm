ICbeReferenceDataServiceGetStations Method

This call is part of reference data and is available to all retail channels.

Get Stations List provides the requester with a list of stations and a limited amount of associated data. The station details in this call includes the necessary basic information for use in Journey Planning and ticket purchased.

Additional station information, including TOC specific information can be provided to front end channels using FEWS 10.7 Get Station Information.

The source data for this FEWs is the IPTIS Stations reference data.

ASSUMPTIONS

  1. It is assumed FEA will keep Stations list in a cache to lookup CRS and NLC codes used in journey planning calls responses.
  2. It is responsibility of the FEA to keep a cache up-to-date i.e. by synchronizing the data on daily basis.
