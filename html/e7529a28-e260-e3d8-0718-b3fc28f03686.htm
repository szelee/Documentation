Central Booking Engine

The main purpose for the Central Booking Engine (CBE) is to provide highly available, scalable and flexible standardized SOAP interface for client applications to the rail industry interfaces. The requirements for the CBE also include performance, resilience and security.
Please note that WebService Contracts namespace contain API that is exposed to the external vendors. Infrastructure namespaces represent mostly lower-level common (shared) structures used within Webservice Contracts.
