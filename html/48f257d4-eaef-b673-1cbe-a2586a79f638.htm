CbeReferenceDataServiceGetSupplements Method

This call is part of reference data and is available to all retail channels. It is used to obtain list of supplements with additional details. The source data are from IPTIS reference data SupplementDefinitions.

Remarks

BACKGROUND

To be used by FEA to display description rather than codes to end-users, used in FEWS 2.2 response.

In some cases description may not be present but the short description is. For this reason both fields are marked as optional so that they may or may not have data associated with them.

BUSINESS PROCESS CONTEXT

Not required.

PROCESS DIAGRAM

None provided.

PROCESS DESCRIPTION

None provided.

ASSUMPTIONS

  1. It is assumed FEA will keep Supplements list in a cache to lookup supplement codes used in journey planning calls.
  2. It is responsibility of the FEA to keep a cache up-to-date i.e. by synchronizing the data on daily basis.

VALIDATION

None.

Business Validation:

None.
