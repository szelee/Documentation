ICbeReferenceDataServiceGetTicketRestrictions Method

This call is part of reference data and is available to all retail channels. It is used to obtain list of ticket restrictions with additional details. The source data are from IPTIS reference data TicketRestrictions, with supplementary information provided by NRE.

Namespace:  Vix.Cbe.WebService
Assembly:  Vix.Cbe.WebService (in Vix.Cbe.WebService.dll) Version: 0.1.0.0
Syntax
C#
Copy
[OperationContractAttribute]
[FaultContractAttribute(typeof(CbeFault))]
[FewsAttribute("10.11", FewsRevisionNumber.First, "3.0")]
Task<TicketRestrictionMessageResponseV1> GetTicketRestrictions()

Return Value

Type: TaskTicketRestrictionMessageResponseV1
The ticket restrictions reference data.
Remarks

BACKGROUND

Not available.

BUSINESS PROCESS CONTEXT

Not available.

PROCESS DIAGRAM

Not available.

PROCESS DESCRIPTION

Not available.

ASSUMPTIONS

  1. It is assumed FEA will keep Ticket Restrictions list in a cache to lookup codes used in journey planning calls responses.
  2. It is responsibility of the FEA to keep a cache up-to-date i.e. by synchronizing the data on daily basis.

VALIDATIONS

    1. None.
  • Business validation:
    1. None.
